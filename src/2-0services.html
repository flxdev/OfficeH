2
Проектирование Вашего офиса
  • Предпроектные исследования.
  • Планировочные решения (предоставление бесплатных пробных планировок).
  • Разработка Эскизного проекта.
  • Подбор мебели и материалов для реализации решений, указанных в Эскизном проекте.
  • Разработка Рабочего проекта.
  • Разработка плана и графика работ по реализации проекта.
  1. Предпроектные исследования:
    • изучение инженерно-технической документации;
    • изучение объекта с выездом на место;
    • обмеры объекта с привязкой к инженерным коммуникациям;
    • показ иллюстративных материалов и обсуждение стилистики офиса.
  2. Разработка базового ПЛАНИРОВОЧНОГО РЕШЕНИЯ:
    • функциональное зонирование пространства;
    • разработка планировочной структуры помещения (план стеновых перегородок, концептуальный план расстановки мебели и оборудования);
    • экспликация помещений.
  3. Разработка ЭСКИЗНОГО ПРОЕКТА, который включает в себя:
    • утвержденный вариант планировочного решения;
    • детализированный план расстановки мебели и оборудования;
    • разработка объемно-графических, цветовых и фактурных вариантов художественного решения офисного пространства;
    • визуализация офисного пространства.
  4. Подбор мебели и материалов для реализации решений, указанных в эскизном проектировании:
    • подбор мебели от ведущих европейских поставщиков;
    • подбор светильников от ведущих европейских поставщиков;
    • подбор решений по устройству перегородок: ГКЛ, перегородки остекленные, комбинированные, из закаленного стекла, мобильные.
    • подбор решений по дверям: из закаленного стекла, глухие, в алюминевом профиле, противопожарные, противодымные и т.д.
    • подбор решений для потолка: ГКЛ, модульный потолок, акустические панели, грильято.
    • побор решений для пола: фальшпол, ковровое покрытие (плитка, рулонное), гетерогенное, паркет, плитка
    • подбор решений для оформления окон: роллеты, жалюзи, шторы
    • подбор электротехнического оборудования;
    • подбор решений по автоматизации и безопасности офиса: IT инфраструктура, «умный офис», автоматизация переговорных, конференц и презентационных комнат, охрана, сигнализация, биометрика.
  5. Разработка РАБОЧЕГО ПРОЕКТА:
    • план помещения после перпланировки с размерами, экспликакция помещенй;
    • план расстановки мебели и оборудования;
    • план размещения санитарно-технического оборудования с привязками;
    • план размещения выключателей, розеток и электровыводов;
    • план потолка с указанием типа используемого материала;
    • план с привязкой осветильных приборов;
    • план пола с указанием отметки уровня пола, типа напольного покрытия, его рисунка и размера, включая экспликацию напольного покрытия с указанием площади и артикула выбранного материала;
    • спецификация дверных блоков;
    • разрезы и развертки стен с декоративными элементами;
    • ведомость отделки стен помещения;
    • спецификация мебели и оборудования;
    • чертежи мебели индивидуального изготовления.
  6. Разработка плана и графика работ по реализации проекта
